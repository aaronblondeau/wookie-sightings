# wookie-sightings

This is an example app which demonstrates how to add custom field editors to Kottster.

## Getting started

Install dependencies

```
pnpm install
```

Setup database (note DB_FILE_NAME in .env)

```
npx drizzle-kit push
```

Run Kottster

```
pnpm dev
```
